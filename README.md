# calc_size

calculate the file or folder size

## Usage

```shell
calc_size --help
```

## Example

```shell
calc_size a.txt . .. D:/Code/ /path/not/exist
```

```shell
a.txt            :   1.00  B
.                :  12.00 MB
..               :  10.00 GB
D:/Code/         :  12.00 GB
/path/not/exist  :  -1.00  B
```
